# ProtonPasswordMerger
A simple utility to merge duplicated password for Proton Pass.
I was trying to move to ProtonPass, but had many passwords scattered across different services and browser, many of them duplicated. so i ended up importing all of the in PP and discovered there's no automatic merge.

# Usage
- export you passwords from PP as a ZIP
- <img width="149" height="356" alt="image" src="https://github.com/user-attachments/assets/e400db8f-bb15-47f5-94e2-72b8b4d7ab36" />
- <img width="703" height="466" alt="image" src="https://github.com/user-attachments/assets/15ae9bd5-cf68-470e-ac16-71a95a905808" />
- open the exported ZIP, extract the data.json somehwere
- start PPM, select the data.json file
- click "merge"
- click "save"
- replace the data.json file in the zip
- optional: delete all passwords in PP, or move them to a different vault
- import the zip in PP
